# User Guide

## Features 

### Adding tasks

There are 3 types of task that you can add.

* ToDo "todo [task description]"
* Deadline "deadline [task description] /by [date in a yyyy/MM/d HHmm format]"
* Event "event [task description] /from [start] /to [end]"
